## CozyHome
Full Stack Airbnb Clone with Next.js 14 App Router: React, Tailwind, Prisma, MongoDB, NextAuth 

# Setup .env file :
```
DATABASE_URL=
AUTH_SECRET=
GITHUB_ID=
GITHUB_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=
```

# Setup Prisma
```
npx prisma db push
```

# Run the app
```
npm run dev
```
